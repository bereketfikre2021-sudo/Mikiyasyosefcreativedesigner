## ğŸ¨ Customization

### Colors
The color scheme is defined in CSS custom properties at the top of `styles.css`:
```css
:root {
    --primary-color: #00d4ff;
    --secondary-color: #ff6b6b;
    --accent-color: #4ecdc4;
    /* ... more colors */
}
```

### Content
- **Personal Information**: Update the name, title, and description in `index.html`
- **Profile Image**: Replace the placeholder image URL with your own
- **Projects**: Modify the `projects` object in `script.js` to showcase your work
- **Social Links**: Update the social media links in the contact section

### Styling
- **Typography**: Change fonts by updating the Google Fonts import
- **Animations**: Adjust timing and effects in the CSS animation properties
- **Layout**: Modify grid systems and spacing as needed
